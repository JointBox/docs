Your DIY smart home. Simplified.

Setup with no coding

The main concept around JointBox is semantic configuration which allows to connect and configure hardware without writing code. Interaction between modules connected to JointBox is configuration driven as well.

Getting Started

Popular modules supported

JointBox supports tons of widely available modules and sensors. Most of the Arduino modules useful for home automation are already supported, more are coming.

DS18S20, Relays, DHT-11, and other already available.

See the list

Network connected

JointBox supports TCP\IP network stack and popular high level network protocols like HTTP and MQTT which enables it to talk to other pieces of home automation software e.g. OpenHab.

About

JointBox is an open-source non-commercial product. It was started by Dmitry Berezovsky as a hobby project when he was building his own smart home.

Legal note

Code is licensed under GPL v3, feel you free to use in your own DIY projects.

Contributions

At this point the only person working on the project is the original author. Since he is not able to dedicate too much time on this contributors are highly required. There are a few fields where you can help:

  • Adding new modules (new hardware support)
  • Implementing drivers for various platforms
  • Sharing your experience of using JointBox with community (tutorials, blog posts, etc.)
